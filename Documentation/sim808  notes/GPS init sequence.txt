POWER ON GPS
AT+CGPSPWR=1    
Example:
AT+CGPSPWR=1
OK


SET GPS Cold mode
AT+CGPSRST=0
Example
AT+CGPSRST=0
OK


GET GPS STATUS
AT+CGPSSTATUS?
Example
AT+CGPSSTATUS? 
+CGPSSTATUS: Location 3D Fix

OK

GET GPS LOCATION
AT+CGPSINF=0
Example
AT+CGPSINF=0
+CGPSINF: 0,4927.657900,1106.062000,316.700000,20220806163428.000,0,12,0.037040,195.630005

OK

POWER OFF GPS
AT+CGPSPWR=0
Example
AT+CGPSPWR=0
OK
